Este proyecto presenta mi portafolio personal, donde se muestran algunos de los trabajos y proyectos desarrollados durante mi formación como Front-End Developer. El objetivo del portafolio es reflejar mi evolución, mi estilo visual y mis conocimientos en diseño y desarrollo web.
El portafolio incluye secciones dedicadas a la presentación personal, habilidades técnicas, proyectos destacados y medios de contacto.  
El diseño busca transmitir una identidad visual coherente y una navegación fluida en todos los dispositivos.

Tecnologías utilizadas:
- HTML5
- CSS3
- JavaScript (ES6+)  
- Git y GitHub
- Figma** (para el diseño y prototipado)
Enlace al diseño en Figma: https://www.figma.com/proto/PaGAi1jXnkBzhuf716rLD6/Project-2---Portfolio?node-id=0-1&t=TqybyHaASheEhaAO-1

Instrucciones para visualizar el proyecto:
1. Clona este repositorio o descárgalo en tu ordenador.  
2. Abre el proyecto en Visual Studio Code (u otro editor).  
3. Instala la extensión **Live Server** si deseas visualizarlo localmente.  
4. Haz clic derecho sobre el archivo `index.html` → selecciona **"Open with Live Server"**.  
5. También puedes abrir directamente el archivo `index.html` en tu navegador.
